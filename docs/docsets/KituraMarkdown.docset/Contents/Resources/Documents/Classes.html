  • KituraMarkdown

    An implementation of Kitura’s TemplateEngine protocol. In particular this templating engine takes files in Markdown (.md) format and converts them to HTML. In addition this class has some helper methods for taking Markdown formatted text and converting it to HTML.

    Note

    Under the covers this templating engine uses the cmark C language reference implementation of Markdown.
